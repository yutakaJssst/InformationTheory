### A.1 対話型AI（汎用）

| ツール名 | 開発元 | URL | 料金 | 得意分野 |

## 付録B プロンプトテンプレート集

本付録では、本書で紹介したプロンプトのエッセンスをテンプレートとして整理する。各テンプレートには、ROCF（Role, Output, Context, Format）の各要素がどこに対応するかを注釈として示している。

> **ROCF要素の凡例**:
> - 【R】Role（役割）: AIに与える役割設定
> - 【O】Output（出力）: 求める出力の内容
> - 【C】Context（文脈）: 背景情報や条件
> - 【F】Format（形式）: 出力の形式指定

### B.1 文献調査用プロンプト

**テンプレート1: サーベイの起点を見つける**

```
【R】あなたは[分野名]の研究者で、文献調査の専門家です。

【O】以下のテーマに関するサーベイ論文・レビュー論文を探すための
検索戦略を提案してください。

【C】
- テーマ: [テーマ]
- 知りたいこと: [具体的な問い]
- すでに知っていること: [既知の情報]

【F】
- 推奨する検索キーワード（英語）5セット
- 推奨する学術データベース
- 注意すべき関連分野
```

**テンプレート2: 論文の要約を依頼する**

```
【R】あなたは学術論文の読解に精通した研究アシスタントです。

【O】以下の論文情報を、構造化された要約にまとめてください。

【C】
- タイトル: [タイトル]
- 著者: [著者]
- 概要: [Abstract または要点を貼る]

【F】
以下の形式で出力してください。
- 研究の目的（2文以内）
- 手法の概要（3文以内）
- 主な結果（箇条書き3点）
- 限界・今後の課題（2点）
- 自分の研究との関連（※ここは自分で記入すること）
```

**テンプレート3: 研究動向の整理**

```
【R】あなたは[分野名]の動向に詳しい研究者です。

【O】以下のキーワードに関する研究動向を整理してください。

【C】
- キーワード: [キーワード1], [キーワード2], [キーワード3]
- 期間: 過去[5]年間
- 対象: [学会名/ジャーナル名]での発表・掲載論文

【F】
- 主要な研究潮流（3〜5つ）と各潮流の代表的な研究
- 最近のトレンド・転換点
- 未解決の課題・今後の方向性
※ 挙げられた論文は必ず実在を確認すること
```

**テンプレート4: 論文間の関係性の整理**

```
【R】あなたは文献レビューの専門家です。

【O】以下の複数の論文の関係性を整理してください。

【C】
論文1: [タイトル, 著者, 年] — [要点]
論文2: [タイトル, 著者, 年] — [要点]
論文3: [タイトル, 著者, 年] — [要点]
...

【F】
- 各論文の共通点と相違点を表形式で
- 研究の発展の流れ（時系列）
- 未解決の問題として残っているもの
```

**テンプレート5: 批判的読解の支援**

```
【R】あなたは厳格な査読者です。

【O】以下の論文の主張について、批判的な視点からの質問を生成してください。

【C】
- 論文の主張: [主張を2-3文で記述]
- 使用された手法: [手法の概要]
- 実験の規模: [データ数、被験者数等]

【F】
以下の観点から各3問ずつ:
- 方法論的な妥当性に関する質問
- 結果の解釈に関する質問
- 一般化可能性に関する質問
```
