# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This repository contains course materials and student submissions for "情報論" (Information Theory), a graduate-level course at Nihon University, College of Science and Technology, Department of Applied Information Engineering. The course is taught by Professor Yutaka Matsuno (松野 裕) and targets M1 students in Information Science, Electronic Engineering, and Precision Machinery Engineering.

## Repository Structure

- **Lecture slides** (`情報論{1-5}_2025.pptx`): PowerPoint slides for each lecture session
- **Lecture PDFs**: Supplementary materials including AI tool workshops, Git/GitHub tutorials, and research presentations
- **Student submission folders** (6 directories):
  - `GitHubを使ったホームページ作成/` — GitHub Pages homepage creation assignments
  - `ガクチカの作成/` — "Gakuchika" (extracurricular achievement) writing with AI
  - `スライドの作成/` — Research presentation slides created with AI tools
  - `英語論文作成/` — English paper translation using AI
  - `文献調査/` — Literature survey using AI
  - `翻訳したい日本語論文を提出してください/` — Source Japanese papers for translation

## Course Theme

The course focuses on using AI tools (Claude, ChatGPT, DALL-E, Midjourney, Gamma, Tome) for graduate research activities: literature surveys, slide creation, English paper writing, GitHub/web publishing, and career preparation.

## Key Notes

- This is a document-only repository (PDFs, PPTX, DOCX) — no source code to build, test, or lint.
- Student submissions contain personally identifiable information (names, student IDs). Handle with care.
- Professor Matsuno's research specialty is D-Case/Assurance Cases, system safety, and dependability (particularly in autonomous driving).
- The `.gitattributes` uses LF normalization for text files.
- Large binary files (PowerPoint, PDF) are stored directly in git (no LFS configured).
